Detailed Description

A genetic Deque class. It can be thought of as a double-ended queue, hence Deque. This allows for an O(1) AddFront, AddBack, RemoveFront, RemoveBack. The Deque also has O(1) indexed lookup, as it is backed by a circular array.

Based on https://unitylist.com/p/cxl/Web-Rtc-Unity-Plugin-Sample
